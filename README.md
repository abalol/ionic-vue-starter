# Ionic/Vue Starter

## initialize
```
ionic capacitor add ios
ionic capacitor add android
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Build For XCode
```
npm run ios
```
